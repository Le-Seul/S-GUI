S-GUI

The Small Graphical User Interface

View on GitHub Documents Download .zip Download .tar.gz

S-GUI是一个面向小型嵌入式系统的轻量级图形用户界面库，它是一个使用C语言开发，支持多任务环境，基于窗口的图形用户界面库。使用S-GUI能方便地开发人机界面，用户因此可以不用花太多精力去处理界面繁杂的逻辑细节，取而代之的是较为简单的“事件驱动模式”。

目前，S-GUI可以在STM32等32位MCU上移植并支持彩色LCD，不过，S-GUI对LCD的接口没有限制，只要驱动程序能够实现像素的写入和读取功能即可移植。S-GUI支持硬件图形加速，在STM32F429等带有图形加速单元的单片机上运行时可以有效降低CPU的占用量。
