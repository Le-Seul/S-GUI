# S-GUI
* This is a Small Graphical User Interface.
* 这是一个小型的用户图形界面.

S-GUI是为单片机等小型嵌入式系统设计的图形用户界面库.这个工程是S-GUI在Windows下的模拟器例子，工程使用Visual Studio 2015编译，S-GUI/S-GUI/GUI目录下是S-GUI库的代码，您可以把它移植到特定的平台，比如单片机系统。作者也已经成功把S-GUI移植到STM32F429I DiscoVery开发板并实现了类似模拟器中的例子，关于移植事项请咨询作者。

详情请参看S-GUI文件夹下的[ReadMe.txt](https://github.com/Le-Seul/S-GUI/blob/master/S-GUI/ReadMe.txt)文件.
